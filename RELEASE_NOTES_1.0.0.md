# Smart Bookmark v1.0.0 Release Notes

**发布日期**: 2026-05-08  
**最后更新**: 2026-05-08 (修复版本)

## 产品特性

### 核心功能
- ✨ **智能书签分类**: 使用 AI 自动分类和组织书签
- 📁 **文件夹管理**: 创建、重命名、删除文件夹和子文件夹
- 🔍 **快速搜索**: 在新标签页中搜索和浏览书签
- ⚡ **快捷键保存**: 使用快捷键（Alt+Shift+S）快速收藏当前页面
- 💾 **备份恢复**: 保存和恢复书签快照

### 智能整理功能
- 🎛 **可配置粒度**: 支持粗略、均衡、精细三种分类粒度
- 🌐 **多语言命名**: 支持中文、英文、自动检测
- 📊 **分类依据**: 按主题、场景、内容类型、平台分类
- 📝 **自定义指令**: 支持自定义分类偏好
- 👀 **整理预览**: 应用前预览整理结果
- 🔄 **环形进度条**: 实时显示整理进度百分比

### AI 提供商支持
- Claude (Anthropic)
- DeepSeek
- Gemini (Google)
- 智谱 GLM
- 通义千问 (阿里)
- Kimi (月之暗面)

### 国际化支持
- 🇨🇳 简体中文
- 🇨🇳 繁體中文
- 🇺🇸 English

## 最新改进 (v1.0.0)

### UI 优化
- ✅ 环形进度条替代数字显示，更美观直观
- ✅ 百分比进度显示，一目了然
- ✅ 固定圆形外观，避免 loading 状态被压扁

### 可靠性修复
- ✅ 改进 AI 分类的 JSON 解析，支持 AI 添加解释文本的情况
- ✅ 智能提取 JSON 数组，提高响应处理的鲁棒性
- ✅ 国际化文案补全（granularity 和 classifyBy 提示文字）
- ✅ 更优的错误日志，便于问题诊断

## 技术实现

- **Manifest V3**: 完全符合 Chrome 最新扩展标准
- **React 19**: 现代化前端框架
- **TypeScript**: 类型安全开发
- **SVG 环形进度条**: 使用 SVG circle 绘制动态进度
- **Chrome Storage API**: 安全的数据存储
- **Chrome Bookmarks API**: 原生书签管理

## 包内容

- `dist/` - 完整的打包发布文件
- `manifest.json` - 插件配置文件
- `icons/` - 插件图标资源
- `assets/` - 编译后的 JS/CSS 资源

## 安装步骤

1. 解压 `smart-bookmark-1.0.0.zip`
2. 打开 Chrome 并访问 `chrome://extensions`
3. 启用"开发者模式"
4. 点击"加载未打包的扩展程序"
5. 选择解压后的 `dist` 文件夹

## 已知限制

- 批量分类操作受到 API 速率限制（建议 1000+ 书签分次操作）
- 需要有效的 AI API Key 才能使用智能分类功能
- 某些网站的书签可能由于跨域限制无法访问
